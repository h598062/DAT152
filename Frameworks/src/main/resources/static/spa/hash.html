URLs with hash sign

Open network tab in browser and observe which of the below links that results in a server request.

URLs that do not send request to server:

  • #
  • hash.html#
  • #pageOne
  • hash.html#pageOne
  • #/user/ole
  • hash.html#/user/ole

URL that send do a request to server:

  • hash.html

The last URL aske the web server for document, but browser can probably use the cache to fillfill the request.
