Override toJSON to serialize instance of Class

All output is to the web console only.

Click the button to serialize instance of the following class: 

class Student {
    #name = "Ole Olsen";
    #phone = "12345678";

    constructor(name,phone) {
        this.#name = name;
        this.#phone = phone;
    }

    getName() { return this.#name; }
    setName(name) { this.#name = name; }
    getPhone() { return this.#phone; }
    setPhone(phone) { this.#phone = phone; }

    toJSON() {
        return {
            name: this.getName(),
            phone: this.getPhone()
        }
    }

}
