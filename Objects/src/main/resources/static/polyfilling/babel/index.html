Babel transcompiled JavaScript files

Before the demonstartions can run by Eclipse:

  1. Copy the content of /Objects/src/main/resources/static/class/classdemo to the directory /Objects/src/main/resources/static/polyfilling/babel/.
  2. Transcompile with Babel the content of /Objects/src/main/resources/static/polyfilling/babel/classdemo/js.
  3. Replace /Objects/src/main/resources/static/polyfilling/babel/classdemo/js with the result from from Babel.
  4. Refresh Eclipse to see content copied into the Eclipse workspace.

The Eclipse project must then be refreshed to see the updated content. The below pages should then work also in older browsers.

  • Public fields and public static fields
  • Private fields and static private fields
  • A demonstration with subclass
